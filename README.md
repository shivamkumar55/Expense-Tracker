# 💰 Expense Tracker

A responsive **Expense Tracker** web application built using **HTML, CSS, and JavaScript**. It helps users efficiently manage their income and expenses by tracking transactions, calculating balances, and storing data locally in the browser using the **Local Storage**.

---

## 🚀 Features

- ➕ Add income and expense transactions
- ✏️ Edit existing transactions
- 🗑️ Delete transactions
- 💾 Store data using Local Storage
- 📊 Automatic calculation of:
  - Total Balance
  - Total Income
  - Total Expenses
- 📅 View transaction history
- 📱 Fully responsive design
- ⚡ Simple, clean, and user-friendly interface

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **Local Storage**
